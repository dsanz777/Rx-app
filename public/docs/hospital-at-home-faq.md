## 1. Program Overview
- **Definition:** Acute-level care delivered at the patient’s residence with daily clinician oversight, remote monitoring, and rapid escalation pathways.
- **Ideal diagnoses:** CHF exacerbation, COPD, CAP, cellulitis, low-risk sepsis.
- **Exclusions:** Hemodynamic instability, ICU-level needs, unsafe home, active substance withdrawal, inability to consent.

## 2. Pharmacy Responsibilities
| Task | Detail |
| --- | --- |
| Med verification | IV-to-PO conversion, renal adjustments, deprescribing opportunities |
| Medication kits | Stock antibiotics, anticoagulant reversal, rescue meds with chain-of-custody logs |
| Delivery SLAs | Courier network with <2 hr turnaround + backup vendor |
| Infusion support | Pump competency training + hotline for field nurses |

## 3. Operational Stack
| Component | Requirement |
| --- | --- |
| Remote vitals kit | BP cuff, pulse oximeter, temperature, optional scale |
| Communications | 24/7 command center, secure clinician chat, telehealth platform |
| Transportation | Dedicated courier + EMS / rideshare contracts |
| Documentation | EHR order sets, med pass note, escalation template |

## 4. Sample Daily Workflow
1. **0700** – Command huddle (MD, RN, pharmacist) reviewing census and alerts.
2. **0900** – Nurse visit: assessment, vitals upload, medication administration.
3. **1200** – Virtual physician round; pharmacist available for med adjustments.
4. **1600** – Second visit for wound care/infusion if ordered.
5. **1900** – Remote monitoring check; escalate per ladder if vitals out of range.

## 5. Escalation Ladder
| Tier | Trigger | Action |
| --- | --- | --- |
| Green | Stable vitals | Continue plan, asynchronous review |
| Yellow | Mild deviation (SBP 90–100, SpO₂ 90–92) | Immediate virtual review, optional extra visit |
| Red | Severe deviation (SBP <85, SpO₂ <88 on baseline O₂) | Activate EMS transport to partner hospital |

## 6. Compliance & Billing Notes
- CMS Acute Hospital Care at Home waiver active through 2024 (monitor extensions).
- Document two clinician touchpoints per day (nurse + MD/APP) and all vitals uploads.
- Maintain controlled substance custody logs and patient consent forms.
- Align with payer-specific documentation for Humana, BCBS FL, Medicare Advantage.
