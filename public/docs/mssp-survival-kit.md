## 1. Program Pillars
1. **Attribution & Benchmark Hygiene** — monthly prospective roster review, RAF refresh, AWV automation.
2. **Pharmacy Intervention Layer** — deprescribing, GLP‑1/SGLT2 adoption, infusion and specialty spend oversight.
3. **Fast Escalation Infrastructure** — 24/7 nurse line decision tree, community paramedicine partners, EMS + home care integration.

## 2. Required Infrastructure
| Asset | Description | Owner |
| --- | --- | --- |
| MSSP command dashboard | Real-time attribution, financial “leak” tracker, risk segmentation heat map | Analytics |
| Pharmacy consult line | SLA ≤2 business days, templated consult note in EHR | Clinical pharmacy |
| Transitional care workflow | 48-hour TCM call, med-rec checklist, post-discharge pharmacist touch | Nursing + pharmacy |
| Community resource spine | Transportation, SNF/HHA roster, social support pathways | Care coordination |

## 3. Pharmacy-Led Services
- **48-Hour Post-Discharge Call:** Medication reconciliation, SDOH screen, follow-up scheduling.
- **Polypharmacy review:** ≥12 meds or ≥5 high-risk classes flagged in dashboard.
- **Infusion site-of-care optimization:** Biosimilar conversions, home vs. outpatient infusions, specialty pharmacy oversight.

## 4. Financial Guardrails
| Metric | Target |
| --- | --- |
| SNF average length of stay | ≤20 days |
| ED visits per 1,000 | <150 |
| Pharmacy PMPM variance | ±$90 vs. risk-adjusted benchmark |
| GLP-1/SGLT2 uptake (eligible cohort) | ≥40% |
| AWV completion | ≥75% |

## 5. Communication Toolkit
- **Provider one-pager:** Referral workflow, consult SLA, sample documentation.
- **Patient FAQ:** 24/7 support instructions, medication refill access, escalation ladder.
- **Care coordinator script:** Color-coded decision tree for triage and pharmacy consult triggers.
